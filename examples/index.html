Basic example

This is the simplest example of using pinnd.js. The nav element to the left will float as you scroll down the page. The nav has been instantiated with the following options: 

var opts = {
    selector: '.nav'
}
pinnd(opts);

Because no context is provided for this example, the pinning action defaults to the body element.
