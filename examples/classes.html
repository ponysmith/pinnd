CSS Classes

pinnd.js exposes the following classes for manipulating the pinned elements in their various states:

  • .pinnd: This class will exist on the pinned element if it is pinned to the top of the viewport, or if it is pinned to the bottom of its context element. Basically, if it is in any position except its default position.
  • .pinnd-top: This class will exist on the pinned element when it is pinned to the top of the viewport
  • .pinnd-bottom: This class will exist on the pinned element when it is pinned to the bottom of its context element

Scroll down the page and watch the class indicators below the nav links to see when classes are active.
