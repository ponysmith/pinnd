Basic examples › Using context

Here, we've set a context selector. pinnd will then constrain the pinned element to that container. In this case, the left nav will only pin while within the container. Once the container flows offscreen, the pinned element will fix to the bottom of the container and flow off the screen as well. It's probably easier to visualize this with an example, so just scroll down this page and watch the left navigation item. It will remain within its context.

Note that the selector option will be filtered by the context. In other words, only elements that match the selector string and are children of the context element will be instantiated as pinnd elements. 

var opts = {
    selector: '.nav',
    offset: 20,
    context: '.pinnd-context'
}
pinnd(opts);
