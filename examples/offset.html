Basic examples › Using offset

This example makes use of the offset option. Offset allows you to set the pixel amount to offset the pinnd element from the top of the viewport. Here, we've set an offset of 20 to make the navigation element pin 20px from the top of the viewport. 

var opts = {
    selector: '.nav',
    offset: 20
}
pinnd(opts);

There are a couple important things to note about pinnd.js and the offset option.

  1. if you have the need, you can set a negative offset to make the element pin outside the viewport. Why you'd want to do that, who knows, but you could.
  2. pinnd.js takes padding into account when pinning elements. It does not, however, account for margins. Take this into account or you may see a jumping effect when the element pins.
