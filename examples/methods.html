Callbacks

Need a little more control of when elements do and don't pin? pinnd.js provides public methods to manipulate the pinning action of any set of objects:

  • disable():
  • enable():

When the pinnd() function is called, an object will be returned with the public methods exposed. So all you need to do to take advantage of the public methods is to capture the function call return in a variable (we've used in the sample below.

Here, we're dynamically enabling and disabling the pinning of the headers in the right column. 

var opts = {
    selector: '.content-header',
    context: '.pinnd-context',
}
var headers_pinnd = pinnd(opts);

$('a.status-toggle').on('click', function(e) {
    var enabled = $(this).hasClass('status-enabled');
    var link = $(this);
    if(enabled) {
        headers_pinnd.disable();
        link.removeClass('status-enabled').addClass('status-disabled');
    } else {
        headers_pinnd.enable();
        link.removeClass('status-disabled').addClass('status-enabled');
    }
});
